# Reliable Plumbing Services

A modern, high-converting, and visually stunning 4-page web application designed for a premier professional plumbing company. Built with clean, desktop-first precision and mobile-first responsive architecture.

---

Live Demo: https://plumb-flow-landing-page.vercel.app/

## 🛠️ Tech Stack & Architecture

- **Framework:** React 18+ (Functional components with hooks)
- **Build Tool:** Vite (High-speed development engine)
- **Language:** TypeScript (Strict type-safety & interfaces)
- **Styling:** Tailwind CSS (Utility-first styling with modern fluid spacing)
- **Typography:** Montserrat (Display headings) paired with Inter (Clean body copy)
- **Iconography:** Lucide React (Crisp, vector layout indicators)
- **Animations:** Motion / CSS transitions (Sophisticated micro-interactions)

---

## 🌟 Key Features & Design Choices

### 1. High-Converting Page Architecture
1. **Home:** Dynamic hero showcase, certified trust seals, "Why Choose Us" detailed benefit cards, and live testimonial carousels.
2. **Our Services:** Interactive procedures guide detailing emergency services, hydro-jetting, smart leak detection, and an **interactive, variable-driven cost range estimator tool**.
3. **About Us:** Narrative timeline of the company's master plumbing heritage, safety compliance specifications, and an **interactive 5-digit ZIP code area coverage validator**.
4. **Contact Us:** Fully validated lead intake form with immediate feedback, emergency dispatch toggles, active business hours grid, and vector GPS hotspot maps.

### 2. Micro-Interactions & Styling
- **Sticky Glassmorphic Navigation:** Dynamic backdrop-blur header preserving the "Get Free Quote" CTA in high visibility at all times.
- **Pulsing CTAs & Micro-Scales:** Responsive hover transitions (1.03x scale) for service cards and pulsing interactive buttons to maximize clicks.
- **Real-Time Dispatch Ledger:** A fully featured inspector dashboard tracking captured leads, estimated cost ranges, and active dispatch statuses.

## 🚀 Getting Started

### Installation
Ensure you have Node.js installed, then execute:
```bash
npm install
```

### Run Local Dev Server
Start the high-speed Vite server on the default port:
```bash
npm run dev
```

### Production Build
Compile and bundle optimal static assets inside the `dist/` folder:
```bash
npm run build
```
